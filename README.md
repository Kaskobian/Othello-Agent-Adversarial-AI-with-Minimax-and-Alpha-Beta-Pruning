# Othello-Agent-Adversarial-AI-with-Minimax-and-Alpha-Beta-Pruning
An AI Othello (Reversi) agent using Minimax with Alpha-Beta pruning and iterative deepening to make optimal moves within 30 s. It generates legal moves in row-major (i, j) coordinates, prints board states before/after each move, and features a countdown timer plus single-move undo.
